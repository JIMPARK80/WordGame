# 🎨 Word Game - 그림 맞추기 게임

그림을 보고 단어를 맞추는 웹 기반 게임입니다.

## ✨ 기능

### 게임 모드
- **타이핑 모드**: 그림을 보고 직접 단어를 입력
- **객관식 모드**: 4개의 선택지 중 정답 선택

### 언어 지원
- 한국어 / English
- 실시간 언어 전환 가능

### 게임 기능
- 113개 이상의 단어 데이터베이스
- 힌트 기능
- 점수 시스템
- 실시간 피드백

## 🚀 실행 방법

1. 프로젝트 폴더를 열기
2. `index.html` 파일을 브라우저에서 열기
3. 게임 시작!

## 📁 파일 구조

```
WordGame/
├── index.html      # 게임 HTML 구조
├── style.css       # 스타일시트
├── script.js       # 게임 로직
├── words.json      # 단어 데이터베이스
└── README.md       # 프로젝트 설명
```

## 🎮 게임 방법

1. 언어 선택: 상단의 "한국어" 또는 "English" 버튼 클릭
2. 게임 모드 선택: "타이핑" 또는 "객관식" 버튼 클릭
3. 그림을 보고 정답 입력/선택
4. 힌트 버튼으로 도움 받기 (점수 -2점)
5. 정답 시 +10점 획득

## 📝 단어 추가하기

`words.json` 파일을 편집하여 단어를 추가할 수 있습니다:

```json
{
  "emoji": "🎮",
  "word": {
    "ko": "게임",
    "en": "game"
  },
  "hint": {
    "ko": "재미있는 놀이",
    "en": "Fun activity"
  },
  "image": "game.png"  // 선택사항
}
```
