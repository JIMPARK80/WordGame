# 🎮 Picture Word Game - 게임 전체 요약

## 📖 게임 개요

**Picture Word Game**은 그림(이모지 또는 이미지)을 보고 해당하는 단어를 맞추는 교육용 웹 게임입니다. 두 가지 게임 모드를 제공하며, 한국어와 영어를 지원합니다.

---

## 🎯 게임 방식

### 1. 게임 시작
- **시작 화면**: 게임 시작, 옵션, 게임 종료 버튼 제공
- **옵션 설정**: 게임 모드, 언어, 사운드 설정 가능
- **게임 시작**: "게임 시작" 버튼 클릭 시 랜덤으로 10개의 문제가 선택됨

### 2. 게임 진행

#### 문제 구조
- 각 문제는 **그림(이모지/이미지)**, **힌트**, **정답**으로 구성
- 힌트는 자동으로 표시됨 (점수 감점 없음)
- 각 문제마다 **10초 타이머** 제공

#### 게임 모드

**A. 객관식 모드 (Multiple Choice)**
- 4개의 선택지 중 정답 선택
- 정답 1개 + 오답 3개 (랜덤 배치)
- 버튼 클릭으로 답 선택

**B. 타이핑 모드 (Typing)**
- 그림을 보고 직접 단어 입력
- 대소문자 구분 없음
- Enter 키로 제출 가능

### 3. 점수 시스템

- **정답 시**: +10점
- **오답 시**: 0점
- **타이머**: 10초 제한 (시간 초과 시 자동 오답 처리)
- **점수 누적**: 스테이지 간 점수 유지

### 4. 스테이지 시스템

- **1스테이지**: 10문제
- **스테이지 클리어 조건**: 10문제 모두 정답 (10/10)
- **완벽한 점수 시**: 다음 스테이지로 진행, 스테이지 번호 증가
- **미완성 시**: 같은 스테이지 재시도

### 5. 게임 종료

- 10문제 완료 시 **스테이지 클리어 화면** 표시
- **통계 표시**: 최종 점수, 정답 수, 정확도
- **다음 스테이지** 또는 **다시 시도** 선택 가능

## 🎨 게임 화면 구성

### 시작 화면
- 게임 타이틀: "🎨 Picture Word Game"
- 버튼: 게임 시작, 옵션, 게임 종료
- 배경: 보라색 그라데이션

### 게임 화면
- **상단 정보판**:
  - 점수 (Score)
  - 스테이지 (Stage)
  - 정답 수 (Correct)
  - 문제 번호 (Question: X / 10)
  - 타이머 (Time) - 5초 이하 시 경고, 3초 이하 시 위험 표시

- **중앙 영역**:
  - 그림 표시 영역 (이모지 또는 이미지)
  - 힌트 영역 (자동 표시)

- **하단 영역**:
  - 객관식: 4개 선택 버튼 (2x2 그리드)
  - 타이핑: 입력 필드 + 제출 버튼
  - 피드백 메시지

- **제어 버튼**:
  - 새 게임 버튼
  - 설정 버튼 (톱니바퀴 아이콘)

### 옵션 모달
- **게임 모드**: 객관식 / 타이핑
- **언어**: English / 한국어
- **설정**: 사운드 ON/OFF
- **네비게이션**: 게임 화면으로 / 시작 화면으로

### 스테이지 클리어 화면
- 축하 애니메이션 (완벽한 점수 시)
- 통계 표시: 점수, 정답 수, 정확도
- 다음 스테이지 / 다시 시도 버튼

## 🎨 UI/UX 특징

### 디자인

## 📝 구현 내역 (Implementation Notes)

### 주요 구현 기능

#### 1. Exit 버튼 기능 ✅
- **위치**: 시작 화면의 "게임 종료" 버튼
- **기능**: 확인 대화상자 표시, `window.close()` 시도, 실패 시 안내 메시지
- **개선사항**: `setTimeout`을 사용하여 창이 실제로 닫히지 않았을 때만 메시지 표시

#### 2. 옵션 모달 기능 ✅
- **톱니바퀴 아이콘 클릭**: 게임 중 톱니바퀴 아이콘 클릭 시 옵션 모달 열기
- **배경 투명도**: 90% 투명도 (alpha 0.1), 보라색 계열
- **블러 효과**: `backdrop-filter: blur(2px)` 적용
- **크기**: 최대 너비 500px, 최대 높이 90vh

#### 3. 옵션 모달 내부 기능 ✅
- **게임 모드 선택**: Multiple Choice / Typing 모드 전환 (보라색 배경)
- **언어 선택**: English / 한국어 전환 (초록색 배경)
- **설정**: 사운드 ON/OFF 토글 (분홍색 배경)
- **네비게이션 버튼**: 게임 화면으로, 시작 화면으로 (노란색 배경)

#### 4. X 버튼 효과 ✅
- **호버 효과**: 90도 회전 + 확대, 배경색 밝아짐, 그림자 효과
- **클릭 효과**: 약간 축소
- **펄스 효과**: 호버 시 확장되는 원형 효과

#### 5. 시작 화면 UI 개선 ✅
- **버튼 중앙 정렬**: `align-items: center` 적용
- **버튼 너비**: 80%로 설정
- **버튼 간격**: 30px
- **배경색**: 연한 노란색

#### 6. Favicon 추가 ✅
- **파일**: `favicon.svg`
- **디자인**: 보라색 그라데이션 배경, 팔레트 이모지 (🎨) 아이콘

#### 7. 이모지 자동 생성 기능 ✅
- **자동 매칭 시스템**: 이모지가 없거나 "?"인 경우 자동으로 적절한 이모지 할당
- **200개 이상의 단어-이모지 매핑 데이터베이스** 포함
- **5단계 매칭 로직**: 직접 매칭 → 부분 매칭 → 힌트 기반 → 카테고리 기반 → 기본 이모지

### 스타일 개선사항

#### 색상 테마
- 메인 색상: 보라색 그라데이션 (`#667eea` → `#764ba2`)

- 경고 색상: 분홍색 (`#f5576c`)

- 옵션 모달 배경: 90% 투명도 (alpha 0.1)
- 블러 효과: 2px

#### 버튼 스타일
- 그라데이션 배경
