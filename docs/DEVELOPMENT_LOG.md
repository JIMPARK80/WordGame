# 개발 로그 (Development Log)

## 2025년 11월 28일 (목요일)

### 스코어보드 아이콘 및 애니메이션 추가
- **점수 아이콘**: 별 (⭐) - 금색 배경 (#FFD860), 점수 증가 시 흔들림 애니메이션, 크기 30% 확대
- **스테이지 아이콘**: 트로피 (🏆) - 청록색 배경
- **정답 아이콘**: 행복한 얼굴 (😊) - 초록색 배경, 정답 증가 시 점프 애니메이션
- **문제 아이콘**: 퀴즈북 (📘) - 보라색 배경, 문제 번호 업데이트 시 펄럭임 애니메이션
- **시간 아이콘**: 탁상시계 (🕐) - 빨간색 배경, 3초 이하일 때만 흔들림 애니메이션

### UI 색상 개선 (Calm Kids Set 적용)
- **선택지 버튼**: Soft Lavender Blue (#C9D8FF) 배경, 밝은 블루 테두리 (#7C8FFF)
- **정답 버튼**: Mint Light Green (#A8E6A3)
- **오답 버튼**: Coral Pink (#FF9C9C)
- **피드백 영역**: 버튼 색상과 통일
- **버튼 텍스트**: font-weight 700으로 증가 (아이들용 가독성 향상)

### Options 모달 개선
- **크기 축소**: max-width 500px → 400px, max-height 90vh → 80vh
- **상하 사이즈 축소**: 패딩, 마진, 폰트 크기 등 전체적으로 축소
- **폰트 크기 20% 증가**: 가독성 향상
- **게임 일시정지 기능**: 모달 열 때 타이머 일시정지, 닫을 때 재개

### 코드 최적화
- CSS 변수 도입으로 코드 유지보수성 향상
- 중복 코드 제거 및 정리

## 2025년 11월 27일 (수요일)

### 초기 게임 개발
- **Initial commit**: Word Game with typing and multiple choice modes
- 기본 게임 구조 및 기능 구현

### 게임 기능 추가
- **다국어 지원**: 한국어/영어 전환 기능
- **게임 모드**: 객관식(Multiple Choice) / 타이핑(Typing) 모드
- **스테이지 시스템**: 10문제 = 1스테이지
- **타이머 기능**: 10초 제한, 시간 초과 시 자동 오답 처리
- **자동 진행**: 정답/오답 후 3초 자동 다음 문제 진행
- **축하 효과**: 정답 시 폭죽 애니메이션 (Canvas Confetti)
- **사운드 효과**: 정답/오답/클릭 사운드

### UI/UX 개발
- **시작 화면**: 게임 시작, 옵션, 종료 버튼
- **스테이지 클리어 화면**: 스테이지 완료 시 결과 표시
- **옵션 모달**: 게임 모드, 언어, 사운드 설정
  - 게임으로 돌아가기 버튼 추가
  - 시작 화면으로 돌아가기 버튼 추가
  - X 버튼 효과 (회전, 확대/축소)
  - 섹션별 배경색 구분
- **설정 메뉴 버튼**: 데스크톱용 톱니바퀴 아이콘
- **햄버거 메뉴**: 모바일용 메뉴 버튼
- **반응형 디자인**: 모바일 최적화
  - 햄버거 메뉴 위치 조정
  - 스코어보드 레이아웃 개선
  - 메뉴 패널 간격 최적화
  - 버튼 레이아웃 개선
  - 이미지 표시 간격 최적화

### 스코어보드 개선
- **레이아웃 재구성**: 2행 구조
  - 상단: 점수(Score) / 스테이지(Stage)
  - 하단: 정답(Correct) / 문제(Question) / 시간(Time)
- **스테이지 증가 로직 수정**: 완벽한 점수 시에만 스테이지 증가

### 데이터 및 콘텐츠
- **단어 데이터베이스**: words.json 파일 추가
- **이모지 매칭 테이블**: 이모지 자동 매칭 기능
- **이미지 연결**: 특정 단어에 이미지 파일 연결 완료
- **Favicon 추가**: 게임 아이콘 (SVG)

### 모바일 레이아웃 최적화
- 힌트/피드백 영역 통일 (동일한 크기 및 스타일)
- 한 줄 텍스트 표시 (폰트 크기 자동 조정: 15px~20px)
- 정답 메시지는 항상 20px, 오답 메시지는 유연한 크기
- 빈 피드백 영역 완전히 숨김 처리
- New Game 버튼 제거 (시작 화면으로 통합)
