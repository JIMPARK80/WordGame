## 📝 참고사항

1. **이미지 중복 불가**: 각 이미지 파일은 반드시 하나의 단어에만 연결되어야 합니다. 같은 이미지 파일을 여러 단어에 사용하는 것은 허용되지 않습니다.

2. **의미상 적절한 중복은 유지 (이모지만)**: 예를 들어 🎤 (마이크, 노래하다, 가수)는 모두 음악과 관련되어 있어 이모지 중복이 적절합니다. 하지만 이미지는 각각 고유해야 합니다.

3. **자동 매칭 기능**: `script.js`의 `generateEmoji()` 함수가 이모지가 없거나 "?"인 경우 자동으로 이모지를 할당합니다.

4. **이모지 가용성**: 일부 이모지는 모든 플랫폼에서 지원되지 않을 수 있으므로 테스트 후 사용하세요.
