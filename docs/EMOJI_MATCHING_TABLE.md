## 📝 참고사항

1. **의미상 적절한 중복은 유지**: 예를 들어 🎤 (마이크, 노래하다, 가수)는 모두 음악과 관련되어 있어 중복이 적절합니다.

2. **자동 매칭 기능**: `script.js`의 `generateEmoji()` 함수가 이모지가 없거나 "?"인 경우 자동으로 이모지를 할당합니다.

3. **이모지 가용성**: 일부 이모지는 모든 플랫폼에서 지원되지 않을 수 있으므로 테스트 후 사용하세요.
