# 프로젝트 구조 (Project Structure)

## 📁 핵심 파일

- `index.html` - 메인 HTML 파일
- `script.js` - 게임 로직 및 기능
- `style.css` - 스타일시트
- `words.json` - 단어 데이터베이스 (271개)
- `words_by_level.json` - 레벨별 단어 분류
- `favicon.svg` - 파비콘

## 📁 문서 파일

- `README.md` - 프로젝트 소개 및 사용법
- `STATS.md` - 단어 데이터 상세 통계
- `docs/PROJECT_OVERVIEW.md` - 프로젝트 전체 개요 (통합 문서)
- `docs/DEVELOPMENT_LOG.md` - 개발 로그
- `docs/FUTURE_FEATURES.md` - 향후 기능 제안
- `docs/test-mobile.md` - 모바일 테스트 가이드
- `docs/FONT_SIZES.md` - 폰트 크기 참고 자료
- `docs/EMOJI_MATCHING_TABLE.md` - 이모지 매칭 테이블

## 📁 이미지 파일

- `images/` - 게임에서 사용하는 이미지 파일들
