# 🎮 Picture Word Game - 프로젝트 개요

## 📖 게임 소개

**Picture Word Game**은 그림(이모지 또는 이미지)을 보고 해당하는 단어를 맞추는 교육용 웹 게임입니다. 3세~7세 어린이를 대상으로 한 레벨 기반 학습 게임입니다.

### 핵심 특징
- **레벨 시스템**: 5개 레벨 (3세~7세 연령별)
- **게임 모드**: 객관식(Multiple Choice) / 타이핑(Typing)
- **다국어 지원**: 한국어/영어 실시간 전환
- **오프라인 동작**: 서버 없이 완전 동작

## 🎯 게임 시스템

### 레벨 시스템
- **레벨업 조건**: Perfect 클리어 개수 기준
  - Level 1 → 2: Perfect 3개
  - Level 2 → 3: Perfect 5개
  - Level 3 → 4: Perfect 7개
  - Level 4 → 5: Perfect 10개
  - Level 5: Perfect 10개 유지

### 스테이지 시스템
- **스테이지 클리어**: 정확도 80% 이상
- **Perfect 클리어**: 정확도 100%
- **레벨별 문제 수**:
  - Level 1-2: 3문제
  - Level 3: 5문제
  - Level 4: 6문제
  - Level 5: 7문제

### 게임 모드
1. **객관식 모드**: 4개 선택지 중 정답 선택
2. **타이핑 모드**: 직접 단어 입력

### 점수 시스템
- 정답 시: +10점
- 타이머: 10초 제한
- 자동 진행: 정답/오답 후 3초 자동 다음 문제
